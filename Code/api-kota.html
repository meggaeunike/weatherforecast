Parameter dari API Nominatim

q=query : Free-form query string to search for. Free-form queries are processed first left-to-right and then right-to-left if that fails. So you may search for pilkington avenue, birmingham as well as for birmingham, pilkington avenue. Commas are optional, but improve performance by reducing the complexity of the search.

street=housenumber streetname

city=city

county=county

state=state

country=country

postalcode=postalcode
